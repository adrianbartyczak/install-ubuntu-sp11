================================================================
    Test 1: Playing YouTube for 1 hour and 20 minutes
================================================================

Test:
Play Google Keynote YouTube video at 720p in default (non-theatre) mode in Firefox. Firefox is fullscreen.

Completion date: 07/07/26

Battery start percentage:
70% (starting drain from 75%)

Run time:
1 hour and 20 minutes

Harware configurations:
    WiFi: On
    Bluetooth: Off
    Audio: 0% (Off)
    Screen resolution: 2880x1920
    Screen resolution scale: 175%
    Screen refresh rate: 120Hz
    Screen brightness: 40%

USB attachments:
    None

Ambient temperature (for baseline fan speed control):
80°F

Linux-specific configurations:
    Using the TLP power manager instead of "power-profiles-daemon"
    CPU power governor is set to "schedutil"

--------- Results ----------------------------------------------

Windows Hyper-V running Debian with XFCE:
58%

Ubuntu with XFCE:
50%

Ubuntu with GNOME:
51%

Arch Linux with XFCE (based on this repository: https://github.com/dwhinham/linux-surface-pro-11):
Most likely will be the same as Ubuntu with XFCE when the "schedutil" governor is used (default's to "performance").
